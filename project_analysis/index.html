Project Analysis

Long Island Historic Fencing Society

Site Intent

The purpose of this redesign is to give the Long Island Historical Fencing Society a user friendly web site with appealing visual elements and easy to understand navigation. The site will be cleaned up and the content shifted and spread over mutable pages to encourage user exploration. The home page will be cleaned up and it’s content minimalized to decrease clutter. A proper gallery page will be added to give order to the media presented and increase interactivity with the end user. A video gallery will also be added as well. Examining the code of the website revealed 21 errors with he HTML5 and 4 errors in the CSS 3 coding. It’s the goal of this redesign to ensure there are no errors in the code and it the website is in full compliance.

Goals for Redesign

The main goal of this redesign is to provide lihfs.org with a modern and visually appealing site that is easy to navigate and uncluttered. We not only want users to be able to easily find the information they need, we want to encourage them through planned placement of site resources to navigate through the site and have a pleasant experience doing so. The main focus will be on visuals and better placement of resources throughout the site with interactivity a priority as well.

It is believed that by making these changes the client will increase their web presence and become a stronger force in the greater HEMA community. The improvement of the UI will encourage visitors to navigate through the site and spend more time reading the content. This will serve to improve their ranking and also their standing in the community which they serve.

By focusing on these elements the client will give a content rich experience the its end users with pleasing visuals and intuitive navigation. Increased membership is also a goal and with a greater web presence and smartly laid out content the conversion rate of casual visitors will be increased thus increasing the membership of the client and enhancing the study experience of its membership through a larger, more vibrant student base.
